# מערבל מבחנים 🔀

אפליקציית ווב שמקבלת קבצי מבחן ב־Word‏ (.docx) או PDF עם שאלות אמריקאיות,
מזהה את השאלות אוטומטית (כולל תמונות צמודות לשאלה), ומאפשרת:

- **ערבוב** — יצירת כמה גרסאות מעורבבות להורדה כקבצי Word + תשובון למורה
- **סימולציה** — מבחן תרגול של 10 שאלות אקראיות מהמאגר, עם ציון באחוזים
  בסוף, שנבדק על ידי ChatGPT באופן עצמאי (בלי להסתמך על סימוני קובץ המקור)

אפשר להעלות כמה קבצי מבחן — כל השאלות מצטרפות למאגר אחד.

## הרצה מקומית

```bash
cd web
npm install
echo "OPENAI_API_KEY=sk-..." > .env.local   # נדרש לסימולציה בלבד
npm run dev
```

הפרסור והערבוב רצים כולם בדפדפן; רק בדיקת הסימולציה עוברת דרך endpoint
שרת (`/api/grade`) כדי שמפתח ה־API לא ייחשף לגולשים. **לעולם אל תשימו את
המפתח בקוד או בגיט.**

## פריסה ל־Vercel

1. Import של הריפו מ־GitHub ב־[vercel.com](https://vercel.com/new)
2. **Root Directory**: `web` (‏Framework: Vite מזוהה אוטומטית)
3. ב־Project Settings → Environment Variables להוסיף:
   `OPENAI_API_KEY` = המפתח שלכם
4. Deploy — הפונקציה `api/grade.js` תרוץ אוטומטית כ־Serverless Function

## פורמטים שמזוהים

- **שאלות:** `1.` / `3)` / `שאלה 2:` בתחילת שורה. שורות המשך מצורפות אוטומטית.
- **תשובות:** `א.` `ב.` `ג.` `ד.` או `a.`/`A)` וכו', כולל כמה תשובות באותה שורה.
- **תמונות** (בקובצי Word): תמונה בסמוך לשאלה נשארת צמודה אליה — בערבוב,
  בסימולציה ובקבצים המיוצאים. היא גם נשלחת ל־ChatGPT בבדיקה.
- **סימון התשובה הנכונה** (רלוונטי לתשובון של גרסאות הערבוב; הסימולציה
  לא מסתמכת עליו):
  - תשובה **מודגשת (Bold)** בקובץ Word
  - כוכבית לפני האות: `*ב. פריז`
  - שורת `תשובה נכונה: ב` אחרי השאלה

## פלט הערבוב

- קובץ Word לכל גרסה (`מבחן - גרסה 1.docx` …) עם כיווניות RTL לעברית
- `מבחן - תשובון.docx` — לכל גרסה: מספר השאלה, מספרה במקור, והאות הנכונה
- תשובות כמו "כל התשובות נכונות" / "None of the above" נשארות במקומן
  (אפשר לכבות בהגדרות)

## מגבלות ידועות

- תמונות מזוהות מקבצי Word בלבד; חילוץ תמונות מ־PDF אינו נתמך עדיין.
- PDF עם עברית ב"סדר ויזואלי" (נדיר) עשוי לדרוש תיקון ידני של הטקסט בממשק.
